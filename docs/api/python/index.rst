Python CrewAI API Reference
===========================

The Python CrewAI component provides autonomous AI crews for development tasks.

Overview
--------

The crew_agents package provides:

- **Crews** - Specialized AI agent crews for different tasks
- **Flows** - Orchestrated workflows combining multiple crews
- **Tools** - Utility tools for file operations and more
